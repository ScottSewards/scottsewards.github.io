Albers by Scott S.

Albers is my modern framework for modern browsers. I use custom elements such as code-box and tab-box to define my content. The aim is to limit the use of CSS and JavaScript.

Theme Colour

I use simple black, white and user-defined colour scheme. Change the colour picker to change the user-defined colour.

Change Theme Colour

I always use dark mode. So I use prefered-color-scheme to change between light mode and dark mode. Change your device theme to change theme mode. This may not work for some browsers or devices.
