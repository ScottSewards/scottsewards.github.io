About

Albers is a modern framework for modern browsers. Use existing HTML elements such as to create layouts and cotent versus classes. Don't use classes to define your content, use elements such as code-block and slide-show, complete with attributes and properties.

Albers provides a simple colour scheme: black, white and user-defined. Modify the user-defined colour below.
